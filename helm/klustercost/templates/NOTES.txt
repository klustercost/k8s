  This means PostgreSQL is using an emptyDir volume and ALL
  DATA WILL BE LOST when the PostgreSQL pod restarts, is
  rescheduled, or is upgraded.

  To enable persistent storage, upgrade the release with:

    helm upgrade {{ .Release.Name }} <chart> \
      --set postgresql.pvc.enabled=true \
      --set postgresql.pvc.storageClass=<your-storage-class> \
      --set postgresql.pvc.size=8Gi
